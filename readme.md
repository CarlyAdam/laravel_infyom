# Laravel Infyom Example

Laravel Infyom Example

### Features
```
Laravel 5
Infyom
AdminLTE
Swagger
One to Many Relationship
Crud
Api Crud
Login
```

### Installing
```
1-Git clone
2-Run command "Composer Install"
3-Define your database connection in .env file
4-Run command "php artisan migrate"
5-Run command "php artisan serve"
```
